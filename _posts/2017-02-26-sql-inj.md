```
$sql = "SELECT * FROM member WHERE username='{$username}' AND password='{$password}'";
```


### 背景：今天接手公司的某个项目，忽然发现这条SQL，恍若隔世，啥年代，还有写成这样的SQL。
		
``` php
$username = trim($_GET['username']);
$password= trim($_GET['password']);

$sql = "SELECT * FROM member WHERE username='{$username}' AND password='{$password}'";
$data = pdo::query($sql);
```

果然,构建了注入字符串 
username: fd' or id=86 #
password: 任意

成功以 admin身份进入后台。。。。
